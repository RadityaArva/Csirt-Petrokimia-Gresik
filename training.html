Latihan bagaimana cara hacker memulai semua

Simulasi Serangan Phishing

  • Materi : mengidentifikasi email atau pesan palsu yang mengandung upaya phishing
  • Kesulitan : Mudah

Latihan Brute Force Attack

  • Materi : simulasi serangan brute force dan mempelajari teknik untuk membuat kata sandi yang lebih aman
  • Kesulitan : Sedang

Pemindaian Kerentanan Jaringan (Vulnerability Scanning)

  • Materi : Menggunakan alat seperti Nmap atau Nessus untuk memindai jaringan
  • Kesulitan : Sedang

Pengujian Keamanan Aplikasi Web

  • Materi : mendeteksi celah-celah seperti SQL injection, Cross-Site Scripting (XSS), dan lainnya dengan menggunakan alat pengujian seperti Burp Suite
  • Kesulitan : Menengah

Latihan Social Engineering

  • Materi : Simulasi interaksi dimana peserta harus mengenali dan mengantisipasi taktik rekayasa sosial yang digunakan
  • Kesulitan : Sedang

Kriptografi

  • Materi : mempelajari algoritma enkripsi serta prinsip dasar kriptografi
  • Kesulitan : Menengah

Analisis Lalu Lintas Jaringan

  • Materi : Menggunakan Wireshark atau alat sejenis untuk menangkap dan menganalisis paket data, mendeteksi aktivitas mencurigakan
  • Kesulitan : Menengah

Analisis Malware

  • Materi : menganalisis sampel malware dalam lingkungan terisolasi (sandbox)...
  • Kesulitan : Sulit

Capture The Flag (CTF)

  • Materi : Latihan komprehensif yang menggabungkan berbagai aspek pengujian keamanan...
  • Kesulitan : Sulit
